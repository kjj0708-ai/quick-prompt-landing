개인정보처리방침

최종 업데이트: 2026년 4월 27일

✅ Quick Prompt는 사용자의 개인정보를 수집하지 않습니다. 모든 데이터는 사용자 본인의 기기 및 Chrome 계정에만 저장됩니다.

1. 수집하는 정보

Quick Prompt 확장프로그램은 다음의 개인 식별 정보를 수집하지 않습니다:

  • 이름, 이메일 주소, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 결제 정보
  • 브라우저 방문 기록
  • 사용자 행동 데이터 또는 분석 데이터

2. 저장하는 데이터

확장프로그램은 다음 데이터를 사용자의 기기 및 Chrome 계정에 로컬로 저장합니다:

  • 사용자가 직접 입력한 프롬프트 내용 및 제목
  • 카테고리 이름 및 설정
  • 썸네일 이미지 (사용자가 직접 업로드한 경우)
  • 확장프로그램 환경설정

이 데이터는 Chrome Storage API를 통해 사용자의 기기(로컬) 및 Chrome 계정(동기화)에만 저장되며, 개발자 또는 제3자 서버에는 전송되지 않습니다.

3. 제3자 서비스

Quick Prompt는 광고 배너 표시를 위해 쿠팡 파트너스 API(api-gateway.coupang.com)와 통신합니다. 이 요청은 상품 정보 조회를 위한 것이며, 사용자의 개인 정보는 포함되지 않습니다. 해당 API는 쿠팡의 개인정보처리방침을 따릅니다.

4. 권한 사용 목적

  • storage: 프롬프트, 카테고리 등 사용자 데이터를 기기에 저장하고 Chrome 계정 간 동기화
  • clipboardWrite: 사용자가 요청한 프롬프트 내용을 클립보드에 복사
  • sidePanel: 브라우저 우측 사이드 패널에 확장프로그램 UI 표시

5. 데이터 공유

Quick Prompt는 사용자 데이터를 어떠한 제3자와도 공유, 판매, 임대하지 않습니다.

6. 데이터 보안

모든 프롬프트 데이터는 사용자 본인의 Chrome 계정 또는 로컬 기기에만 저장됩니다. 개발자는 이 데이터에 접근할 수 없습니다.

7. 아동 개인정보 보호

Quick Prompt는 13세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.

8. 개인정보처리방침 변경

본 방침은 변경될 수 있으며, 변경 시 이 페이지에 최신 내용이 게시됩니다. 최종 업데이트 날짜를 통해 변경 여부를 확인하실 수 있습니다.
